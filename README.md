# Download and Installation
- Clone the repo: `$ git clone git@github.com:mdinsaf07/GridSystem.git`
- Open the repo using Virtual Studio Code.
- Run the code via live server.

# Technology Stack 
- HTML
- CSS
- Bootstrap
- JavaScript
